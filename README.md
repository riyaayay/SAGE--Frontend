# Overview

SAGE Frontend is a standalone cinematic web experience designed to visualize a futuristic multi-agent AI system through immersive storytelling, cinematic camera motion, and real-time 3D rendering.

The project focuses entirely on frontend experience design:
- cinematic transitions
- immersive scene composition
- interactive storytelling
- premium visual presentation
- smooth WebGL-powered animation systems

Built completely from scratch using vanilla JavaScript and Three.js without relying on frontend frameworks.

# Running Locally

Clone the repository:

```bash
git clone https://github.com/riyaayay/SAGE--Frontend.git
```

Navigate into the project:

```bash
cd SAGE--Frontend
```

Run with a local server.

Using VS Code:
- Install **Live Server**
- Right click `index.html`
- Select **Open with Live Server**

Or using Python:

```bash
python -m http.server
```
